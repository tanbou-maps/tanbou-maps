application_user のための
自分または他人のプロフィール参照画面
自分のプロフィール編集画面
自分のアカウント(application_user)を削除する仕組み
を制作したい

コントローラーは下記コマンドで作成済み
```
bundle exec rails generate controller profile
```

ユーザーのプロフィール参照画面は
自分または他人の
user_id, nickname, account_type を表示して
active storage を使用したアバター画像(avatar_image), 背景画像(cover_image)
を表示する

自身のプロフィールなら編集画面へ遷移するボタンを表示してその先では
user_id, nickname, email, avatar_image, cover_image
を編集する画面を表示できるようにする

また自身のプロフィール参照画面にはアカウント削除ボタンを表示して、
ユーザーが任意でアカウントを削除する仕組みを作成する

バックエンドのルーティングには REST API の設計思想で 保守管理を用意にしてほしい
